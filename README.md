Prerequisites
---

[Node.js](https://nodejs.org/en/)

Initial Setup
---

### Client

```bash
cd client/
npm uninstall -g create-react-app
npx create-react-app ./
```

### Server

```bash
cd server/
touch index.js
npm init -y
npm install body-parser cors express mongoose nodemon
# import libraries in index.js
```
