Prerequisites
---

[Node.js](https://nodejs.org/en/)

Usage
---

Start the client and server apps in two separate terminals:

```shell
cd client/
npm install
npm start
```

```shell
cd server/
npm install
npm start
```

Use the app in your browser: <http://localhost:3000>
